Le plugin Envoi-sms-cleversmsWP vous permet d'envoyer des SMS depuis l'interface administrateur en personnalisant l'émetteur.

== Description ==

Le plugin Envoi-sms-cleversmsWP vous permet d'envoyer des SMS via la plateforme d'envoi de SMS de Clever Technologies. Elle est accessible à tous les utilisateurs, même pour les non-experts. Vous pourrez bénéficier des fonctionnalités suivantes:

* Envois de SMS unitaires, SMS en nombre ou groupés
* Émetteur personnalisé avec le nom de votre société
* Etre notifié d'un nouveau commentaire par SMS
* Etre notifié d'un nouveau post par SMS
* Envoyer des SMS avec une Url de votre site internet
* Gestion accusés de réception avec état des SMS envoyés
* Gestion du STOP SMS pour la désinscription conforme
* Récupération des Réponses  (MO)
* Compatible tous opérateurs et tous pays
* Possibilité d'adresser des messages longs et concaténés
* Paiement de vos campagnes et recharges par Carte Bancaire
* Envois de SMS en UCS2 et latin (Cyrillique, Chinois, Arabe...)
* Navigation avec le protocole sécurisé Http(S)
* Interface accessible depuis la plate forme.


**Pour utiliser le plugin [Envoi-sms-cleversmsWP](http://cleversmslight.fr/solution-sms/sms-wordpress), il vous faudra créer un compte gratuitement sur la plate forme SMS [CleverSMSLight](https://cleversmslight.clever-is.fr/fr/inscription). Après l'ouverture du compte, vous pouvez recharger du crédit par carte bancaire avec une navigation sécurisée (https) ou faire une demande à notre service commercial via le formulaire de contacts ou bien par téléphone (+33)1.60.53.60.53) pour obtenir 5 SMS de test.**

== Installation ==

L'installation du plugin peut se faire de 2 façons :

* Méthode 1

1. Téléchargez depuis le site `Envoi-sms-cleversmsWP.zip`
2. Uploader le dossier `Envoi-sms-cleversmsWP` dans le répertoire `/wp-content/plugins/`
3. Activer le plugin SMS Envoi-sms-cleversmsWP dans le menu `Extensions` de Wordpress

* Méthode 2

1. Rechercher le plugin "Envoi-sms-cleversmsWP" à partir du menu "Extensions" de Wordpress
2. Lancer l'installation du plugin

**LA CONFIGURATION DU PLUGIN Envoi-sms-cleversmsWP SE FAIT VIA LE MENU  : RÉGLAGE DE L'INTERFACE ADMINISTRATION DE WORDPRESS.**


* Identifiant : correspond à l’identifiant fourni par le système lors de l’ouverture du compte sur la plate forme CleverSMSLight.

* Mot de passe (obligatoire) : correspond au mot de passe fourni par le système lors de l’ouverture du compte sur la plate forme CleverSMSLight.

* Emetteur (Facultatif) : Caractères alphanumériques comme nom de l’expéditeur. Sa longueur maximale est de 11 caractères.

* Téléphone (Facultatif) : Téléphone de l’administrateur du site WordPress. Ce numéro est utilisé pour recevoir les notifications par SMS. format international.

* Notification d’un nouveau commentaire (Facultatif) : Permet de recevoir un SMS lorsqu’un nouveau commentaire est fait sur le blog.

* Notification d’un nouveau post (Facultatif) : permet de recevoir un SMS lorsqu’un nouvel article est posté sur le blog.
